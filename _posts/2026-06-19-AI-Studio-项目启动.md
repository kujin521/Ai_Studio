## 项目简介

AI Studio 是一个利用 AI 技术生成学习文档和总结的个人知识项目。

## 技术路线

1. **AI 生成内容** — 使用 Claude Code / ChatGPT 生成技术学习文档
2. **博客展示** — 通过 GitHub Pages + Jekyll 将文档发布为个人博客
3. **持续积累** — 定期使用 AI 学习新技术并输出总结

## 博客架构

```
GitHub 仓库 (kujin521/Ai_Studio)
       │
       ▼
Jekyll 静态站点生成
       │
       ▼
GitHub Pages 托管
       │
       ▼
https://kujin521.github.io/Ai_Studio/
```
